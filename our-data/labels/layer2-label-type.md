# Layer Two Label Type

Projects building layer two's.

`label_type` = `layer2`

### Label Subtypes

| Subtype            | Description                                                           |
| ------------------ | --------------------------------------------------------------------- |
| `token_contract`   | ERC-20 token (fungible token) contract owned by the layer two project |
| `general_contract` | A contract controlled by the layer two project                        |
